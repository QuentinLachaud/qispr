Very quick and simple free version of wispr flow. 
Not a full application just yet...
I have it aliased locally to run python dictate.py

"Qispr" then listens for left shift + right shift (no need to hold down) 
->  you speak 
-> press left shift + right shift again 
-> copied to clipboard AND pasted in active dialog box if you are currently in one.

A little animation in the top left corner lets you know what stage you are in.
